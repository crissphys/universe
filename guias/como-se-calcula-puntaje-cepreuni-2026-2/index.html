Cómo se calcula el puntaje CEPREUNI 2026-2

El puntaje final no es un promedio simple. Combina prácticas calificadas, exámenes parciales y examen final con pesos distintos.

1. Fórmula general

Para especialidades distintas de Arquitectura, el puntaje se calcula como PID = S + 2EP1 + 4EP2 + 6EF. S suma las mejores cinco notas de las seis primeras prácticas calificadas más la séptima práctica, que no se elimina.

2. Qué nota se elimina

Solo se descarta la menor nota entre PC1, PC2, PC3, PC4, PC5 y PC6. La PC7 permanece porque corresponde a humanidades y entra completa.

Ejemplo

Si las seis primeras PC son 110, 125, 90, 130, 118 y 140, se elimina 90. Luego se suma 110 + 125 + 130 + 118 + 140. Si la PC7 es 122, se añade completa.

3. Nota mínima histórica

Sirve como referencia, no como promesa. La demanda, dificultad del examen y distribución de postulantes pueden cambiar.

4. Recomendación

Después de cada evaluación revisa tu puntaje acumulado y el examen que más peso falta. Si aún falta EP2 o EF, prioriza cursos de mayor arrastre.
