POLÍTICA DE PRIVACIDAD

Qué información utiliza Universe to Study, para qué se usa y qué controles conserva cada usuario.

Datos de cuenta

  • Se utiliza el nombre visible, identificador de cuenta, rol, preferencias y métricas de actividad necesarias para mostrar el perfil.
  • Cuando no existe un nombre, el navegador puede conservar una identificación anónima para mantener continuidad.
  • No se solicita publicar documento de identidad, contraseña bancaria ni información académica sensible.

Identificación técnica

  • La dirección IP puede consultarse únicamente para reconocer la autorización de propietario definida por el sistema.
  • La etiqueta de propietario requiere además un equipo de escritorio autorizado y no se concede automáticamente a otros dispositivos.
  • Otros usuarios solo reciben el rol ordinario mientras no exista una autorización posterior.

Firebase y sincronización

  • Firebase permite sincronizar cuenta, presencia, puntos, rachas, chats y cambios entre las páginas de Universe to Study.
  • Los datos se vinculan mediante identificadores técnicos para conservar la sesión y evitar mezclar cuentas.
  • La disponibilidad y seguridad de la infraestructura también se rigen por las condiciones de Google Firebase.

Chat mundial

  • El nombre visible, rol, fecha y contenido de cada mensaje pueden ser vistos por otras personas conectadas.
  • No debes publicar teléfonos, direcciones, contraseñas, documentos ni datos privados.
  • Los mensajes pueden conservarse para mantener el historial y permitir moderación.

Soporte privado

  • Cada conversación se separa por usuario y solo se presenta al remitente y a la cuenta propietaria.
  • Los mensajes se almacenan para que la conversación continúe desde las distintas páginas.
  • El usuario debe evitar compartir datos que no sean necesarios para resolver su consulta.

Almacenamiento local

  • El navegador guarda tema claro u oscuro, sesión, preferencias, avance y selecciones de juego para mantener la experiencia.
  • Estos datos permanecen en el dispositivo hasta que se limpien desde el navegador.
  • Bloquear el almacenamiento puede impedir que algunas funciones recuerden su estado.

Apuntes y archivos

  • Los archivos cargados para prácticas se procesan principalmente en el navegador.
  • El texto extraído puede conservarse de forma temporal en la sesión o almacenamiento local.
  • El usuario puede cerrar la sesión o limpiar los datos del sitio para retirar esa información local.

Analítica y servicios externos

  • Google Analytics mide visitas y funcionamiento general sin convertir el contenido del chat en una publicación analítica.
  • Drive, WhatsApp, YouTube, juegos WebGL y otros enlaces aplican sus propias políticas al abrirse.
  • Universe to Study no controla los datos que esos servicios solicitan fuera del sitio.

Control, seguridad y cambios

  • Puedes modificar el nombre permitido, cerrar la sesión o eliminar datos locales desde el navegador.
  • Ningún sistema es infalible; se aplican límites de acceso y separación de conversaciones para reducir riesgos.
  • Las actualizaciones importantes de esta política se reflejarán aquí.
